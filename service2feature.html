ViewerPlus Features

FEATURE NAME
DESCRIPTION
INTUITIVE USER INTERFACE
ViewerPlus's intuitive and user friendly interface hosts different panels to display XML backbone in hierarchical tree form to view submission in current, cumulative and sequence.
HIGH PERFORMANCE
ViewerPlus can quickly load and process large eCTD submission, as long as sequences conform to ICH and regional technical specifications.
PREVIEW DOCUMENTS
Using built-in document viewer panel to quickly preview documents, metadata attributes, envelopes (regional information), lifecycle info, STFs, PDF hyperlinks, etc.
NAVIGATE TO DESTINATIONS
Navigate to the destination documents from bookmarks, hyperlinks and annotations while viewing the PDF document.
ADD REVIEW COMMENTS
Add review comments Facility to add and mange comments by submission reviewers while reviewing eCTD or NeeS submission.
PRODUCT UPDATES
Product updates If any software and specification updates, user receives updates automatically without re-installing the software.
SUPPORTED DOSSIER FORMATS
Supports viewing of eCTD submission sequences from agencies US FDA, EU EMA, CA, JP, CH, GCC, AU, ZA, JO, TH and ZA.
