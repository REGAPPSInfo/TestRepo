PDFAssist Features

DOCUMENT OPEN, EDIT AND SAVE

FEATURE NAME
DESCRIPTION
VIEW DOCUMENT
Easily open and view a PDF or non-PDF file from local storage. PDFAssist has built-in PDF viewing capability. Adobe Acrobat is not required to view and process PDF documents. User can easily browse through multiple paged PDF files.
VIEW ENCRYPTED PDF
User can open and view encrypted files, also by applying encryption user can protect PDF files and restrict unauthorized people from viewing or editing them. User can remove encryption from a PDF as well.
SPLIT DOCUMENT
Split PDF document by page range or by page size with different options provided to contain destinations for bookmarks and hyperlinks.
ADD STICKY NOTE
Add sticky notes on selected location on page.
VIEW/DELETE COMMENTS
View list of comments in grid format and selectively delete them using different options provided.
SPLIT DOCUMENT
Split PDF document by page range or by page size with different options provided to contain destinations for bookmarks and hyperlinks.
CHANGE TEXT COLOUR
Change Text Colour Modify and update the selected text colour with-in rectangle.
SELECT TEXT TOOL
Select Text Tool Change cursor to text select tool.
SELECT HAND TOOL
Change cursor to display Hand tool.
DOCUMENT FAVOURITES
Add documents to favourites list for quick access and mange them.
OPEN FILE IN NATIVE APPLICATION
Open non PDF file in its native application.
SAVE DOCUMENT
PDF document can be saved using default save settings set by user. The default save settings are Magnification, Open to page, Page layout, Fast web view, PDF version and Navigation. User can still customize save settings based on their requirements.

PAGE VIEW, NAVIGATION AND MANIPULATION

FEATURE NAME
DESCRIPTION
PAGE HEADER AND FOOTER
Add,Modify or Delete page Header/Footer for selected page(s) with default or customised values .
WATERMARK
Add or delete Watermark for selected page(s) using different options provided.
PAGE BACKGROUND COLOUR
Change background colour for selected page(s) using different options provided.
REDACT CONTENT ON PAGE
Mask the text by applying any colour using different options provided.
ROTATE PAGE(s)
Rotate Page(s) Rotate selected page(s) 90 degrees Clockwise or Counter Clockwise.
DELETE PAGE(s)
Delete selected page(s) from open document.
REPLACE PAGES(s)
Replace single or more pages from source document with selected page(s) from external document .
EXTRACT PAGE(s)
Extract Page(s) Extract selected page(s) from source document to new a new file.
INSERT PAGES FROM FILE
Insert Pages from File Insert a file at selected position in source document.
INSERT BLANK PAGE
Insert Blank Page Insert a blank page in source document at any position.
UPDATE PAGE ROTATION
View all current page rotations in a grid format and update page orientation for selected page(s). User can do batch update.
UPDATE DOCUMENT METADATA
View document metadata (i.e. Title, Subject, Author, Keywords, etc.) in grid format and update them. User can do batch update.
CHANGE PAGE SIZE
Change Page Size Change current page size of selected page(s) to A4 or Letter.
SCALE PAGE CONTENT
Increase or Decrease page content of selected page(s) from open document by X%.
PAGE NAVIGATION
Page Navigation Navigate pages to First Page,Previous Page,Next Page,Last Page,Page N.
DOCUMENT OR PAGE HISTORY VIEW
Previous View, Return to Next View, Previous Document, Next Document.
ZOOM PAGES
Set the page view Zoom level to Actual Size, Fit Width, Fit Height, Fit Visible, Zoom To (x).
SCROLL PAGES
Scroll pages Vertically or Horizontally.
PAGE DISPLAY SPREADS
Display pages in Odd Spreads, Even Spreads and No Spreads.

BOOKMARKS, HYPERLINKS AND ANNOTATIONS

FEATURE NAME
DESCRIPTION
BOOKMARKS
Add/Edit/Delete bookmarks. Create bookmark for destination types Go to a page within same PDF document (GoTo), Go to a page in external PDF document(GoToR), Go to a page in non PDF document(OpenFile) and Go to web URL (WebURL).
HYPER LINKS
Add/Edit/Delete hyperlinks. Create hyperlink for destination types Go to a page within same PDF document (GoTo), Go to a page in external PDF document(GoToR), Go to a page in non PDF document(OpenFile) and Go to web URL (WebURL ).
NAMED DESTINATIONS
Add/Edit/Delete named destinations.

ADD-ON TOOLS FOR eCTD COMPLIANCE

FEATURE NAME
DESCRIPTION
AUDIT LINK AND BOOKMARKS
Validate and report for all Bookmark(s), Link(s), Annotations and Document properties for selected PDF document(s) against ICH regulatory rules and guidelines. Enable user to audit them and fix validation errors.
TOC LINKS AND BOOKMARKS
Create Table of Contents (TOC) links and/or TOC Bookmarks using different options provided.
BOOKMARK PROCESSOR
An advanced bookmark processing tool extracts all bookmark information into grid format and also displays bookmark tree. Enables user to spell check bookmark title, export and manage other details.
VIEW LINK INFORMATION
Mouse hover on links display link information such as Action, Target Page and Magnification.
IMPORT/EXPORT LINKS AND BOOKMARKS
Import/Export hyperlinks and bookmarks in Excel format.
FILE ATTACHMENTS
Add/Remove file as Attachment to open PDF document.
CREATE LINK ON TEXT/AREA
Create hyperlink for selected text or selected area using different options provided.
RETRIEVE AND UPDATE LINK INFORMATION FOR TEXT COLOR/KEYWORD
Retrieve and update hyperlink properties for entered keywords or by searching text colour using different options provided.
CREATE HYPERLINK FOR KEYWORDS
Create hyperlink for Keywords Create hyperlinks for one or more matching keyword(s) for selected page range using different options provided.
CREATE BOOKMARKS FROM LINKS
Create bookmarks from using different system settings provided.
DELETE ALL/HIGHLIGHTED LINKS
Delete All/Highlighted Links Delete hyperlink(s) using different options provided.
QUICK DELETE LINK(s) AND BOOLMARKS(s)
Delete hyperlinks and bookmarks quickly using different options provided.
BATCH PROCESSING
Batch processing of PDF documents supported for all add-on tools.
