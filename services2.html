Overview of ViewerPlus

ViewerPlus by REGAPPS, is a Windows application for Viewing and Validating electronic submissions prepared in eCTD, NeeS and other electronic dossier formats. It is fast, user friendly and easy to use application. It efficiently utilizes system resources while loading large eCTD submissions.

To process and load a lifecycle of an application, eCTD submission sequences (documents and folders) need to be available through the local or network shared file system

ViewerPlus works on any Windows version and runs on laptops, desktops or server.
